# Glosario

| Término | Definición |
|---|---|
| **Bono político** | Deuda que el Estado reconoce a un partido por gastos de campaña. En VELAR es un token on-chain. |
| **TSE** | Tribunal Supremo de Elecciones: emite y audita los bonos. |
| **Partido** | Tenedor del bono; puede transferirlo o publicarlo al marketplace. |
| **Escrow** | Depósito en garantía on-chain (Trustless Work) que coordina una compraventa. |
| **Soroban** | Plataforma de contratos inteligentes de Stellar. |
| **Classic Asset** | Activo nativo de Stellar (no-Soroban) usado para representar el bono. |
| **Trazabilidad** | Historial completo y verificable de un bono: emisión, dueños, transferencias. |
| **audit_events** | Tabla que registra cada acción crítica del sistema para auditoría. |
| **CountryProfile** | Configuración por país (reglas, moneda, compliance) del modelo multi-país. |
