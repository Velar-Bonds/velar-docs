# Multi-país

VELAR está diseñado con una dimensión de país y un `CountryProfile` por jurisdicción.

- **Costa Rica** — soporte completo (100%).
- **Colombia / Brasil / Argentina** — en roadmap (perfiles y reglas de compliance).

Un guard de compliance evita operaciones cross-border indebidas. En modo demo
(`NEXT_PUBLIC_DEMO_MODE=1`) se muestra un selector de país para ilustrar la
adaptabilidad; el usuario real queda fijado a su jurisdicción.
