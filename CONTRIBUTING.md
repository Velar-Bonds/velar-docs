# Contribuir a la documentación

La documentación vive en `docs/` en formato Markdown.

## Cómo agregar o editar una página

1. Creá/editá el `.md` correspondiente en `docs/`.
2. Si agregás una página nueva, enlazala desde el índice del `README.md`.
3. Commit convencional: `docs: <qué cambiaste>`.
4. Abrí un PR contra `main`.

## Estilo

- Español neutro, frases cortas, ejemplos concretos.
- Usá tablas para comparar y bloques de código para comandos.
- Un H1 por página; subtítulos con H2/H3.
